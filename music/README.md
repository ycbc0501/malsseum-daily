# music/

Hymn / worship instrumental tracks for the daily Reels. The poster rotates
through these one per day (by date).

## Add tracks here (one-time)

Drop **3–5 MP3 files** of royalty-free hymn / worship instrumental music into this
folder. Where to get clean, commercial-use, no-attribution music:

- **Pixabay Music** — https://pixabay.com/music/  → search "hymn", "worship",
  "peaceful piano", "gospel instrumental" → Download. (Free, commercial use OK,
  no credit required — Pixabay Content License.)
- **YouTube Audio Library** — studio.youtube.com → Audio Library → filter by
  "Attribution not required".

Pick calm, reverent instrumentals (piano / strings / organ). 14s+ is fine —
the video uses the first ~14 seconds and fades out.

> ⚠️ Licensing: traditional hymn *melodies* are often public domain, but a
> specific *recording* has its own copyright. Only use recordings explicitly
> licensed for free commercial use (like the sources above).
